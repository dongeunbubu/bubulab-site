HOW · 재사용 설계

이 데모가 실제로 도는 코드

단지명 한 줄 → 실데이터 카드. 위 트레이스의 실제 호출 순서예요
# 단지명 한 줄 → 실거래 카드까지 (의사코드 · 위 4단계와 동일) region = get_region_code("강남구 개포동") # → LAWD_CD 11680 apts = get_apt_list("11680") # → 단지 232개(5페이지)에서 kapt = match(apts, name="개포자이프레지던스") # → 단지코드 A10023348 # ③ 단지 기본·면적·생활점수 (100% 자동) info = get_apt_info("A10023348") # 3,375세대·GS건설·2023 # ④ 구·월 실거래 표본 → 단지명으로 필터 (반자동) deals = [] for ym in ["202604", "202605"]: rows = get_apt_price("11680", ym) # 월 약 10건 표본 deals += [r for r in rows if r.name == "개포자이프레지던스"] # → 84㎡·132㎡ 3건 추출. 없는 평형은 추측 금지 → '데이터 없음'

SUMMARY

이 데모가 보여주는 것

🔎

검색 = 4단계 해소

단지명 → 법정동(LAWD) → 단지코드(kapt) → 기본정보+실거래. 위 트레이스가 그 실제 순서예요.

🏢

기본정보는 100% 자동

세대수·준공·시공사·주차·면적·생활점수는 단지코드만 있으면 공공데이터로 즉시 채워집니다.

📭

없으면 ‘없음’이라 적기

표본에 안 잡힌 평형·달은 추측하지 않습니다. 정직한 빈칸이 가짜 숫자보다 낫습니다.
